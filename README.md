# Kodi Indonesia

Kumpulan link streaming TV Indonesia untuk Kodi.

## Cara Menggunakan

1. Buka Kodi
2. Pastikan **IPTV Simple Client** sudah terinstall
3. Pilih Configure di IPTV Simple Client
4. Add **Add add-on configuration**
5. Masukan URL M3U nya
